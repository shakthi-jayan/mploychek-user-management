# MPloyChek User Management Portal

## Features
- Role based login
- Admin dashboard
- User dashboard
- Add users
- MongoDB integration
- Secure password hashing using bcrypt
- REST API with Express.js
- Angular frontend
- Vercel deployment

---

## Tech Stack
- Angular
- Node.js
- Express.js
- MongoDB Atlas
- Vercel
